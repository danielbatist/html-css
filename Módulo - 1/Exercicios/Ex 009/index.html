Trabalhando com Listas

Listas Ordenadas

As listas ordenadas são utilizada pela tag <ol> para fazer uma organização por meio de uma lista, que pode ser enumerada,para começar com números,letras e números romanos, sendo eles maiusculas e minusculas, como por exemplos:

  1. Acordar
  2. Ligar para o trabalho
  3. Tomar Café
  4. Escovar os dentes

  2. Ligar para o trabalho
  3. Tomar Café
  4. Escovar os dentes

Listas não ordenadas

As listas não ordenadas são utilizadas pela tag <ul> para classificar uma lista sem ter uma ordem por números ou letras, elas fazem a marcação da lista por meio de símbolos como circulos,discos e quadrados.

  • Leite
  • Tomate
  • Manteiga
  • Arroz
  • Feijão
